# Legal_Firm_Analytics

This project demonstrates **data analytics for law firms**.  

**Objectives:**  
- Track **cases handled by law firms**  
- Monitor **lawyer workload and department efficiency**  
- Analyze **revenue per case and client**  
- Provide insights for **management and strategic decision-making**  

**Skills demonstrated:** SQL, Revenue & Case Analytics, Lawyer Performance, Department Analytics, Dashboard-Ready Queries
